Aqui está o completo C# exemplos de código para o cmdlet de exemplo StopProc04. Este é o código para o `Stop-Process` cmdlet descrito [adicionando conjuntos de parâmetros para um Cmdlet](../cmdlet/adding-parameter-sets-to-a-cmdlet.md).
