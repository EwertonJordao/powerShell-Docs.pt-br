# <a name="getproc02-code-samples"></a>Exemplos de código GetProc02

Aqui estão exemplos de código para o cmdlet de exemplo GetProc02. Esse é o `Get-Process` cmdlet de exemplo descrito em [adicionando parâmetros essa entrada de linha de comando de processo](../cmdlet/adding-parameters-that-process-command-line-input.md). Isso `Get-Process` cmdlet recupera os processos com base em seu nome e, em seguida, exibe informações sobre os processos na linha de comando.

> [!NOTE]
> Você pode baixar o C# arquivo de origem (getproc02.cs) para esse cmdlet Get-Proc usando o Microsoft Windows Software Development Kit para Windows Vista e o .NET Framework 3.0 Runtime Components. Para obter instruções de download, consulte [como instalar o Windows PowerShell e o Download do SDK do Windows PowerShell](/powershell/developer/installing-the-windows-powershell-sdk).
>
> Os arquivos de origem baixado estão disponíveis na  **\<amostras do PowerShell >** directory.
