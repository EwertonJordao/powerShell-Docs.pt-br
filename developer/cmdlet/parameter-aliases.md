## <a name="defining-parameter-aliases"></a><span data-ttu-id="4c4c0-116">Definir Aliases de parâmetro</span><span class="sxs-lookup"><span data-stu-id="4c4c0-116">Defining Parameter Aliases</span></span>

  set { userName = value; }
