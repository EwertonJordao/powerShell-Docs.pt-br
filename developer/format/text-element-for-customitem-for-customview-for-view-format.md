# <a name="text-element-for-customitem-for-customview-for-view-format"></a>Elemento Text para CustomItem para CustomView para View (formato)

Especifica o texto que é adicionado aos dados que é exibido pelo controle, como um rótulo, colchetes para colocar os dados e espaços para recuar os dados. Esse elemento é usado ao definir uma exibição de controle personalizado.

Elemento (formato) elemento ViewDefinitions (formato) modo de exibição (formato) do elemento elemento CustomControl (formato) CustomEntries elemento de configuração para CustomControl para elemento de exibição (formato) CustomEntry CustomEntries para elemento CustomItem de exibição (formato) CustomEntry para o elemento de texto de exibição (formato) para CustomItem (formato)

## <a name="syntax"></a>Sintaxe

```xml
<Text>TextToDisplay</Text>
```

### <a name="attributes"></a>Atributos

Nenhum.

### <a name="child-elements"></a>Elementos filho

Nenhum.
